# TechCart - Full Stack E-commerce Application

## Overview
TechCart is a full stack e-commerce application built using Django REST Framework for the backend and React for the frontend. It provides core e-commerce functionalities such as product management, cart operations, and order handling.

## Tech Stack
- Backend: Python, Django, Django REST Framework
- Frontend: React.js
- Database: MySQL

## Project Structure
techcart-fullstack/
│  
├── backend/     # Django REST API  
├── frontend/    # React Application  

## Features
- User authentication (login and signup)
- Product and category management
- Add to cart functionality
- Order management system
- RESTful API integration

## Current Status
The project is under development with planned improvements:
- Payment gateway integration
- Deployment
